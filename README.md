# rockgarden_booking
south korea rockgarden golf club auto booking code.

### Install firefox geckodriver
     please install the latest geckodriver (https://github.com/mozilla/geckodriver/releases)
  
### install packages
     pip install selenium    
     pip install pyautogui    
     pip install pause    
     pip install telepot
